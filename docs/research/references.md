# Research References

This file records the reusable bibliography for the LMS research base. Entries are organized by where they first became relevant in the project notes. Links point to publisher, DOI, PubMed, arXiv, or stable catalog pages where available.

## How To Use This File

- `Used in`: project notes or principles that currently depend on the source.
- `Claim supported`: the specific project-level claim the source is being used to support.
- `Status`: `resolved` means the bibliographic entry has enough detail for project use; `needs-book-reference-check` means it should later be checked against the book's final reference list for exact formatting; `unresolved-book-bibliography` means the visible author/year in Kindle could not be mapped reliably to a full source without the book's bibliography; `project-local-working-draft` means the source is locally available for internal project use and should be checked for updates before formal citation.
